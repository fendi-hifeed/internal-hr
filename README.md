# HR Presensi — Sistem HR Presensi Berbasis Geolocation & Selfie

## Tech Stack
- **Frontend:** Next.js 16 + TypeScript + Tailwind CSS v4 + shadcn/ui + Recharts
- **Mock Data:** Mock API dengan seed data realistis
- **Map:** Leaflet + OpenStreetMap (geo-location display)

## Modules
- `hr` — Karyawan: Clock In/Out, Leave Request, Personal Dashboard
- `admin` — Admin HR: Company Dashboard, Leave Types, Geo-zones, Employee Management

## Quick Start
```bash
npm install
npm run dev   # http://localhost:3001
```
